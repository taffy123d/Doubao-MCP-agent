# 网络搜索技能

## 功能描述

实时网络搜索功能，通过 DuckDuckGo 免费搜索引擎 API 获取互联网上的最新信息。

## 使用场景

### ✅ 适用场景

- **实时资讯**：查询最新新闻、时事动态
- **技术查询**：搜索编程文档、教程、API 参考
- **百科知识**：查找概念解释、历史背景、人物传记
- **生活信息**：搜索生活常识、健康知识、旅游攻略
- **产品信息**：查找产品评测、价格对比、用户评价

### ❌ 不适用场景

- **数学计算**：需要计算的问题请使用 `calculator` 工具
- **天气查询**：查询天气请使用 `get_city_weather` 工具
- **本地文件操作**：搜索本地文件内容不在支持范围内

## 参数说明

| 参数 | 类型 | 必填 | 默认值 | 说明 |
|------|------|------|--------|------|
| `query` | string | 是 | - | 搜索关键词，支持中英文 |
| `num_results` | int | 否 | 5 | 返回结果数量，范围 1-10 |

## 使用示例

### 基础搜索

```python
# 查询 Python 最新版本
web_search(query="Python 最新版本", num_results=3)

# 搜索新闻
web_search(query="2026年科技新闻", num_results=5)

# 技术文档搜索
web_search(query="FastAPI 教程", num_results=3)
```

### 多轮对话示例

```
用户：帮我搜索一下 Python 3.13 的新特性
助手：【调用了工具：web_search】

🔍 搜索「Python 3.13 新特性」的结果：

【摘要】
```

## 返回结果格式

搜索结果包含以下信息：

- **摘要**：主要答案的简短描述
- **来源链接**：答案的原始来源 URL
- **相关结果**：多条搜索结果，每条包含标题和链接

## 异常处理

| 错误类型 | 处理方式 |
|----------|----------|
| 网络超时 | 返回「搜索超时，请稍后重试」 |
| HTTP 错误 | 返回具体错误码 |
| 其他错误 | 返回简化的错误信息 |

## 技术实现

- **搜索引擎**：DuckDuckGo Instant Answer API（免费、无需 API Key）
- **超时设置**：10 秒
- **最大结果数**：10 条

## 注意事项

1. 搜索结果可能包含过时信息，建议多个来源交叉验证
2. 搜索速度取决于网络环境
3. 某些特定领域的专业信息可能搜索效果有限
4. 搜索关键词越具体，结果越准确
